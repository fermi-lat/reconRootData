reconRootData

This library contains the definition of all ROOT classes contained in GLAST reconstruction ROOT files.

Versions:   

  • v1     First version to explicitly use ROOT v3r0p6

    • NOTE:  there were no changes to the reconRootData code to use Root v3r0p6, however, TBEvent did change, and reconRootData links to the TBEvent library.  Starting with this version of reconRootData, one should not use a version of TBEvent < v2r1.

  • v0r1 Last version to explicitly use ROOT v2r25

  • v0    Initial import of the new package
